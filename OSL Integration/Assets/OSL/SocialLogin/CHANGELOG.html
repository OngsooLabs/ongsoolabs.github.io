Changelog

All notable changes to this package are documented here. The format is based on Keep a Changelog.

[1.1.0] - 2026-06-16

Added

  • Discord Sign-In Support (Zero-SDK): Integrated a pure HTTP loopback auth provider with port fallback support (probing ports 50000-50009), cryptographic CSRF protection, and automatic timeout handling.
  • Centralized editor configuration in SocialLoginSetup.cs for enabling Discord Sign-In (OSL_DISCORD define symbol) and setting credentials.

Changed

  • Fast Enter Play Mode Support: Removed static instance backing fields in SocialLoginSettings to comply with Unity Project Auditor rules.
  • package.json version bumped to 1.1.0 and description updated to include Discord support.
  • Documentation updated (README.md and sociallogin.md) with comprehensive Discord integration and loopback redirect setup guidelines.

[1.0.1] - 2025-06-10

Added

  • UPM compatibility configuration (package.json).
  • Packaging guide for UPM installation.

Fixed

  • Google standalone PC local loopback handling to properly process redirect callback under high-latency network conditions.

[1.0.0] - 2025-01-01

Added

  • Initial release.
  • Unified social login contract interfaces (ISocialLoginProvider and ISignin).
  • Platform authentication adapters: Google (GProvider), Facebook (FBProvider), Apple (AProvider), and Steam (StProvider).
  • Centralized configuration UI via the SocialLoginSetup editor window.
  • Soft-dependency compilation architecture with assembly definitions using conditional define constraints.
