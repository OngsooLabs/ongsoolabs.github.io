Changelog

All notable changes to OSL Deterministic Core are documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

[1.0.0] - Initial Release

Added

  • Core RNG
    • IRandomEngine integer-first PRNG contract with Span<byte> state save/load.
    • PCG32 default engine (PCG-XSH-RR 64/32) with unbiased range generation.
    • PCG32State portable 16-byte little-endian snapshot.
    • OSLRandom developer-facing facade (Range, Chance, etc.).
  • Shuffling
    • OSLDeckShuffler.Shuffle with IList<T> and Span<T> overloads, including partial-range support.
  • Gacha
    • OSLGachaTable<T> immutable weighted draw table with cumulative weights and binary search.
    • GachaEntry<T>, GachaTableJson, OSLGachaTableAsset for code / JSON / ScriptableObject workflows.
    • Pity-aware RollMany overload.
  • Pity
    • IPitySystem pluggable contract with Span<byte> state IO.
    • SimpleHardPity reference implementation (consecutive-miss counter).
  • Combat
    • OSLCombatResolver with integer-first probability APIs (RollPermille, RollProbability, RollDice, RollCritical).
  • Samples
    • Lockstep Demo: split-screen OnGUI simulation with per-tick FNV-1a state hash and SYNC / DESYNC banner.
  • Tests
    • EditMode tests covering determinism, state round-trips, range bounds, distribution sanity, shuffle correctness, pity progression, gacha JSON round-trips, and combat boundaries.
  • Packaging
    • UPM package.json, English README, English LICENSE, QuickStart guide.

Notes

  • Target framework: .NET Standard 2.1.
  • Minimum Unity version: 2021.3.
  • Zero third-party dependencies. Zero allocations on RNG hot paths.
