Changelog

All notable changes to this package are documented here. The format is based on Keep a Changelog.

[1.0.2] - 2026-06-22

Added

  • Discord Token Verification: Added server-side endpoint and cryptographic validation support for Discord authentication codes.
  • Apple JWKS Caching: Implemented a 12-hour in-memory cache for Apple's JSON Web Key Sets (JWKS) to avoid frequent external API calls.
  • Steam Ticket Transmission Safety: Escaped special characters in Steam session tickets via Uri.EscapeDataString to prevent payload corruption during HTTP transmission.
  • Discord Redirect Validation: Validates the OAuth redirection parameters against the server settings to prevent redirect manipulation.
  • Automated Source Packaging: Integrated editor script hook (OSLAssetExporter) to automatically zip the .NET backend project (TokenVerificationServer) into Server_Dotnet_Source.zip upon package export.

Changed

  • Updated package description to reflect official support for Discord token verification.
  • Standardized namespaces and client-side HTTP request payloads to ensure clean integration.

[1.0.1] - 2026-05-27

Changed

  • Bumped com.unity.ugui package dependency version from 1.0.0 to 2.0.0.
  • Cleaned up verification documentation files.

[1.0.0] - 2026-05-12

Added

  • Initial release of the OSL Token Verification System.
  • Standalone .NET Web API boilerplate (TokenVerificationServer) featuring ready-to-use validation controllers for Google, Facebook, Apple, and Steam.
  • Client-side validation helper class TokenVerifyManager for easy HTTP integration inside Unity.
