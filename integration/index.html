Asset Ecosystem

The Ongsoo Labs integration framework is designed so each module works together seamlessly. Browse the asset guides below by category.

  • Deterministic: Zero-dependency, zero-allocation deterministic RNG suite
  • EasyPatcher: Runtime resource patching and integrity validation
  • SocialLogin: Social platform authentication (Google, Apple, and more)
  • IapClient: In-app purchase client module
  • SecureSave: Local data encryption and secure storage
