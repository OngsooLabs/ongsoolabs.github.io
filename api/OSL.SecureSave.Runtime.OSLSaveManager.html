Class OSLSaveManager

Namespace
OSL.SecureSave.Runtime

Client-side facade for the OSL Secure Save system. All write operations are routed through the 'osl_master_save' Cloud Code endpoint, which performs authoritative validation.

NOTE:

  • This wrapper assumes UnityServices.InitializeAsync() and AuthenticationService sign-in have already been performed by the host application before any OSLSaveManager call is made.
  • The compile guard OSL_UGS_CLOUDCODE is expected to be defined once the UGS Cloud Code package (com.unity.services.cloudcode) is present in the project.
