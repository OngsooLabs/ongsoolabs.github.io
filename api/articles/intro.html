Introduction to OngsooLabs API

Welcome to the official documentation for the OngsooLabs API ecosystem.

OngsooLabs API is an enterprise-grade backend platform designed as a comprehensive Knowledge Ingestion Gateway. Our architecture isolates distinct backend services—each highly specialized in data extraction, processing, and delivery—to provide robust, scalable solutions for modern AI and enterprise applications.

Our Vision

In the rapidly evolving landscape of AI, LLMs, and Retrieval-Augmented Generation (RAG), the quality and structure of data are paramount. Our ultimate vision is to create a centralized AI Knowledge Platform.

We aim to seamlessly gather, understand, structure, optimize, and deliver knowledge from a wide variety of heterogeneous sources—including web pages, PDF/Office documents, GitHub repositories, Notion workspaces, and YouTube transcripts. By providing pristine, AI-ready data pipelines, we empower developers to build smarter, more reliable AI systems without worrying about data preprocessing.

Platform Architecture

To achieve massive scalability and maintainability, the OngsooLabs API ecosystem is built upon a strict multi-service architecture:

  • Isolated Microservices: Each service is developed in an isolated environment within a monorepo, strictly adhering to Clean Architecture principles (Domain, Infrastructure, WebApi) with zero-dependency core layers.
  • Independent Scaling & Deployment: Services are fully containerized and deployed with their own independent CI/CD pipelines, ensuring that heavy computational tasks (like web rendering) do not bottleneck lightweight data processing requests.
  • Shared Platform Services: Common technical utilities (such as token counting, text chunking, and text slicing) are decoupled into shared containerized platform services. They communicate securely over private internal networks via lightweight HTTP/JSON or gRPC.

Available APIs

Our platform is continuously expanding. Currently, we offer the following public-facing services:

  • AI Reader API: Our flagship data ingestion service. It acts as an intelligent pipeline that fetches web pages, strips away boilerplate noise, and converts the core content into pure, token-optimized Markdown ready for LLMs.

Looking Forward

As we progress through our strategic product roadmap, you can expect new APIs and features tailored for intelligent caching, automated batch crawling, change detection, and seamless enterprise workspace integrations.
