Introduction to OngsooLabs API

Welcome to the official documentation for the OngsooLabs API ecosystem.

OngsooLabs API is a suite of high-performance, containerized microservices designed to bridge the gap between unstructured data and modern AI applications. Built on robust technologies like .NET, Docker, and AWS, our APIs provide reliable and scalable solutions for data preprocessing, intelligent web scraping, and text manipulation.

Our Vision

In the era of Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG), the quality of your input data is just as important as the model itself. Our mission is to provide developers with seamless pipelines that transform messy, unstructured web content into clean, structured, and AI-ready formats.

Key Services

Currently, the OngsooLabs API ecosystem includes:

  • AI Reader API: Our flagship data ingestion service. It acts as an intelligent proxy that fetches web pages, strips away boilerplate noise (like ads, sidebars, and navigation), and converts the core content into pure, token-optimized Markdown.
  • TextProcessor: A dedicated microservice for advanced text chunking, token estimation, and manipulation, working in tandem with the AI Reader to handle massive documents efficiently.

Architecture Highlights

  • Microservices Architecture: Our backend is decoupled into independent Docker containers (e.g., Scraper and TextProcessor), allowing them to scale independently based on computational load.
  • Performance First: Advanced techniques, such as continuous Browser Context Pooling, ensure that web rendering is ultra-fast with minimal memory overhead compared to traditional headless browser solutions.
  • Enterprise-Grade Routing: Built-in proxy rotators dynamically switch between Datacenter and Residential proxies to maximize collection success rates.

Getting Started

To explore what our API can do, check out the AI Reader API Documentation or visit our RapidAPI Hub to grab an API key and start making requests immediately.
