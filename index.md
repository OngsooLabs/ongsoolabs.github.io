# Ongsoo Labs 통합 프레임워크 문서

Ongsoo Labs의 유니티 에셋 통합 프레임워크 공식 기술 문서입니다. 
본 웹사이트는 DocFX 기반으로 구성되어 최신 API 레퍼런스와 통합 연동 가이드를 제공합니다.

## 에셋 생태계 (Ecosystem)

Ongsoo Labs의 통합 프레임워크는 개별 모듈들이 유기적으로 결합하여 강력한 시너지를 내도록 설계되었습니다. 아래에서 각 카테고리별 에셋 문서를 확인하세요.

- [EasyPatcher](articles/easypatcher.md): 런타임 리소스 패치 및 무결성 검증
- [SocialLogin](articles/sociallogin.md): 구글, 애플 등 소셜 플랫폼 로그인 연동
- [IapClient](articles/iapclient.md): 인앱 결제 클라이언트 모듈
- [SecureSave](articles/securesave.md): 로컬 데이터 암호화 및 보안 저장소

## 시스템 요구 사항
- Unity 2021.3 LTS 이상 권장
- .NET Standard 2.1 호환 환경
