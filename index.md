# Ongsoo Labs Integration Framework Documentation

Ongsoo Labs' official technical documentation for the Unity asset integration framework.
This website is powered by DocFX and provides the latest API references and integration guides.

## Asset Ecosystem

The Ongsoo Labs integration framework is designed so each module works together seamlessly.
Browse the asset guides below by category.

- [EasyPatcher](articles/easypatcher.md): Runtime resource patching and integrity validation
- [SocialLogin](articles/sociallogin.md): Social platform authentication (Google, Apple, and more)
- [IapClient](articles/iapclient.md): In-app purchase client module
- [SecureSave](articles/securesave.md): Local data encryption and secure storage

## System Requirements
- Unity 2021.3 LTS or later (recommended)
- .NET Standard 2.1 compatible environment
