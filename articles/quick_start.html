Quick Start

This guide helps you set up Ongsoo Labs packages quickly in a Unity project.

Prerequisites

  • Unity 2021.3 LTS or later
  • .NET Standard 2.1 compatible project settings
  • Basic familiarity with Unity package and assembly definition workflows

Installation

  1. Import the required Ongsoo Labs package(s) into your project.
  2. Open Unity and allow script compilation to complete.
  3. Resolve any package dependency prompts shown by Unity.

Assembly Definition (asmdef) Setup

  1. Select your project's .asmdef file.
  2. Add references to the required OSL assemblies for the module you are integrating.
  3. Save and recompile.

Verify the Setup

  1. Open a sample scene (if provided by the package).
  2. Enter Play Mode and confirm there are no missing reference or compile errors.
  3. Check the Console for runtime initialization logs.
